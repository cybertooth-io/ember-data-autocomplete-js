ember-data-autocomplete-js
==============================================================================

Ember Data driven `autocomplete.js` text input.


Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above


Docs
------------------------------------------------------------------------------

[View the docs here](https://cybertooth-io.github.io/ember-data-autocomplete-js/).
